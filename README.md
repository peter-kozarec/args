# Introduction
Argument parsing library

## Requirement
You'll need:

* "CMake >= 3.19".

## Build
To build the C++ project, as usual:
```sh
cmake -S. -Bbuild -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```
